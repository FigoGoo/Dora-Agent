本次从本地 `main` 新建 `codex/user-frontend-pages` 独立 worktree，目标是在不影响管理端分支的前提下，先落地用户端首页第一版视觉与交互效果。品牌名称已确认使用 `DORAIGC`。

## 目标

- 使用用户确认的“折叠播放三角 + 中央创作星芒”作为首版可替换 Logo 方向。
- 从 Logo 提炼用户端主题色，并在首页中以 CSS token 方式落地。
- 接入用户确认的新版位图 Logo：去除图片烘焙棋盘底，生成透明 PNG，并统一用于侧栏、登录弹窗和浏览器标签页。
- 实现 `CreationLanding` 首页：左侧导航、ContextHeader、Prompt 创作入口、热门 Skills、最近项目、首屏分类承接和精选作品。
- 覆盖未登录创作动作：保留 Prompt 或触发动作，弹出登录弹窗。
- 优化第二版视觉细节：窄侧栏上下结构、暗玻璃快捷卡和作品卡、登录弹窗分区布局、公开作品预览弹窗。
- 建立卡片信息层级：常规标签使用低视觉重量透明胶囊，重点系统状态使用炫彩胶囊，卡片操作通过 hover/focus 呼出玻璃操作层。
- 建立精选作品素材规格：覆盖 `1:1`、`3:4`、`9:16`、`4:3`、`16:9` 五类常用宽高比，并提供 MV、文旅、电商广告、电影海报、动漫短剧等 AIGC 主题素材。
- 将浏览器页签标题更新为 `DORAIGC 人人都是艺术大师`，首页主标题更新为 `Dora Agent - 人人都是艺术大师`，并移除 `AI creation agent` 说明标签。
- 首页卡片、胶囊、浮层、弹窗等视觉容器均改为无真实边框，改用透明玻璃底、阴影和亮度层次表达结构。
- 精选作品卡片标题统一高亮白色；作者昵称不拼接点赞量；用户头像使用本地随机卡通头像，和作者/标题两行文字等高。
- 精选作品左上角分类标签只显示文本，不再使用分类 icon；hover/focus 后仍在右上角浮出静音和全屏图标。
- 热门 Skill 卡片支持 hover/focus 下弹出 Skill 展示浮层，包含预览图、作者、标题、说明和能力标签。
- 提供桌面和移动端可预览页面，并通过组件测试、构建和浏览器截图验证。

## 非目标

- 不接入真实登录、项目创建、上传、点赞、积分或 AG-UI 接口。
- 不定义新的 API、RPC、AG-UI 字段。
- 不实现完整 Agent 工作台、项目页、资产库或作品详情页。
- Logo 是首版可替换资产，不作为最终品牌定稿。

## 页面行为

- Prompt 输入框支持输入、字数计数和创作工具入口展示。
- 首页首屏保持创作入口为主：Hero、Prompt Composer、热门 Skills 和最近项目组成第一焦点。
- 首屏背景使用从左侧射入的双层柔和灯束，形成方向明确的舞台打光效果，避免整片泛色背景。
- 最近项目在桌面宽屏下撑满主内容区，六个项目卡随屏幕宽度放大，避免居中窄容器造成卡片过小。
- 首屏底部加入 `DORAIGC TV` 分类桥，承接精选作品筛选，并固定露出一段作品封面作为下滑预览，避免首屏和精选作品区明显割裂。
- 开始创作、上传素材、使用 Skill、积分、通知、非首页导航和登录入口均触发登录弹窗。
- 登录弹窗采用品牌侧栏 + 意图预览分区布局，保留触发动作和 Prompt，表示登录后继续原动作。
- 精选作品支持公开预览；预览内可基于作品方向继续创作，随后保留 Prompt 并进入登录承接。
- 精选作品采用铺满主视口的固定列宽 masonry；同一断点下所有卡片等宽，素材宽高比至少覆盖方形 `1:1`、竖版 `3:4`、故事版 `9:16`、横版 `4:3`、宽屏 `16:9`。
- 弹窗支持关闭按钮、点击遮罩关闭和 `Escape` 关闭。
- 顶部不提供深浅色主题切换，减少首屏工具噪音；页面固定使用暗色沉浸式主题。
- 热门 Skills 和精选作品卡片默认只展示核心信息、低权重透明标签和简要数据；具体操作在 hover/focus 时以玻璃操作层浮出，不撑开卡片尺寸。
- 热门 Skills 的 hover/focus 浮层向下弹出，展示该 Skill 的视觉预览、作者、用途说明和标签，帮助用户在点击前判断能力适用场景。
- 精选作品封面左上角展示小型黑玻璃分类标签；hover/focus 后右上角显示静音和全屏播放小图标，底部向外呼出“查看”和喜欢操作。
- 精选作品标题、作者和头像放在封面外部，标题保持白色高亮，作者低权重显示且不跟随点赞数量，让封面素材成为主视角。
- 精选作品标题使用专用 `.work-card__title` 类，保持 `#fff`、13px、700 字重和无光晕纯白文字，不使用 `text-shadow`。
- 精选作品作者头像使用本地卡通头像，并与作者/标题两行文字的整体高度对齐。
- 精选作品分类标签仅保留文本，不使用 icon，避免小尺寸标签视觉噪音。
- 顶部招募状态属于高关注系统标签，使用 logo 主题色的炫彩胶囊突出注意力。
- 精选作品仅展示公开快照模拟数据，不展示源会话、提示词、积分成本或用户隐私。
