## 本次接入补齐项

- 通用资源页支持自定义筛选、表单动作、`PATCH`/`PUT`、JSON 字段、JSON 字符串字段、数组字段、复选框和动作成功提示。
- OpenAPI 同步管理端实际 DTO：管理员创建、用户关键词、系统 Skill 草稿/测试结果、模型供应商、模型价格快照、Tool 策略/计价/白名单、积分发放过期时间、精选作品筛选。
- 业务服务补齐用户关键词搜索、模型价格快照写入和公开作品管理端筛选。
- 高风险操作继续通过确认弹窗、预览 token、原因和幂等键进入审计链路。
