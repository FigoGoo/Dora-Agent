## 测试边界

本文件只验证管理端服务端能力。后台页面表格、Drawer、Modal、Toast、图标、视觉样式和浏览器交互不在测试范围内；涉及确认弹窗的需求只验证 preview/confirm API、`X-Admin-Reason`、幂等键、审计和业务状态变化。

## 风险和待确认

- 兑换码导出结果的交付方式需实现阶段明确；用例只要求服务端导出动作幂等、审计和脱敏。
- 后台首页摘要字段需以 OpenAPI 最终 DTO 为准，但不得突破私有创作内容边界。
- Tool 影响范围 preview 的精确字段需与 Tool/Skill 契约对齐后补充 fixture。

## 验收标准

- 管理端所有高风险写操作均覆盖 `X-Admin-Reason`、幂等、审计和权限。
- 管理端普通用户 token、用户端 cookie 或企业身份不能访问任何后台能力。
- 后台响应、审计和日志均不泄露密钥、完整兑换码、用户私有创作内容或供应商原始响应。
- 下架、禁用、积分发放、模型/Tool/Skill 配置变更均有状态变化和审计证据。
- 管理端测试不包含页面表格、Modal、Drawer 或浏览器点击验证。
