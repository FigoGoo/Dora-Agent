## 背景

C 端页面不只有首页，还包括登录、工作台、项目、资产库、Skill、积分、作品中心、精选作品、个人中心和企业空间。页面模板必须统一，否则即使组件一致，页面仍会出现信息密度、导航位置、空态、响应式和主题风格不一致。

## 目标

- 定义 C 端所有核心页面可复用的布局模板。
- 明确不同页面的主任务、内容区域、状态和响应式规则。
- 保证普通用户和企业用户共用统一工作台，同时保留当前空间感知。
- 保证首页公开作品和精选作品可免登录浏览。
- 统一用户端桌面左右结构，避免不同页面使用不同导航骨架。

## 非目标

- 不定义平台后台页面模板。
- 不定义最终路由名称。
- 不定义 API 字段和权限接口。
- 不做移动 App 原生页面设计。

## 页面壳结构

### 登录后 C 端 App Shell

```text
左侧 AppSideNav
  Logo / 新建创作 / 首页 / 工作台 / 项目 / 资产库 / Skill / 作品中心 / 精选作品 / 积分
  底部：主题 / 用户入口

右侧 AppMain
  ContextHeader：页面标题 / 当前空间 / 积分 / 通知 / 用户
  PageBody：页面主体
  FeedbackLayer：Toast / Modal / Drawer
```

规则：

- 用户端桌面默认左右结构，不使用顶部横向主导航。
- 当前空间必须在右侧 ContextHeader 可见。
- 企业空间和个人空间视觉上必须可区分。
- 积分余额在涉及生成、兑换、消耗页面必须可见。
- 平台后台入口不出现在 C 端普通导航中。
- 左侧 AppSideNav 可折叠为图标栏，但一级入口顺序保持稳定。

### 未登录公开 Shell

```text
左侧 PublicSideNav
  Logo / 首页 / 精选作品 / 公开 Skill
  底部：登录 / 注册 / 主题

右侧 PublicMain
  公开内容区
  登录弹窗触发层
  页脚或轻量说明
```

规则：

- 未登录公开页桌面也采用左右结构，减少登录前后页面骨架跳变。
- 首页公开作品和精选作品支持免登录查看。
- 未登录用户触发点赞、创作、创建项目、上传、查看资产库、积分、通知或下载受限内容时，弹出登录弹窗。
- 登录弹窗需要保留当前页面和动作意图。
- 不展示用户私有入口和企业空间。

## 页面模板类型

| 模板 | 适用页面 | 核心结构 |
| --- | --- | --- |
| `CreationLanding` | 首页、快速创作入口 | AppSideNav + Prompt + Skill + 公开作品 / 最近项目 |
| `AgentWorkspace` | Agent 创作页 | ProjectToolbar + StoryboardPanel + PreviewStage + ChatPanel |
| `GalleryGrid` | 精选作品、个人作品中心 | AppSideNav / PublicSideNav + 筛选 + 响应式作品网格 |
| `LibraryManager` | 资产库、项目/会话、Skill 列表 | AppSideNav + 筛选栏 + 列表/网格 + 详情 Drawer |
| `FormWizard` | Skill Builder、兑换码、企业注册 | 步骤/表单 + 预览/结果 |
| `AccountCenter` | 个人中心、积分中心、企业成员 | AppSideNav + 二级设置导航 + 设置面板 |
| `PublicDetail` | 精选作品详情 | PublicSideNav + 媒体展示 + 作品信息 + 推荐内容 |
| `ErrorPage` | 404、403、500、断网 | 错误说明 + 恢复入口 |

## 首页模板

页面目标：

- 让用户快速开始创作。
- 展示少量热门 Skill。
- 展示公开作品；登录后展示最近项目和精选作品入口。

结构：

```text
左侧 AppSideNav / PublicSideNav
右侧首页内容
  Prompt 输入框
  热门 Skill
  公开作品
  登录后最近项目
  精选作品入口
```

规则：

- 首页首屏不做 Dashboard。
- 首页不展示完整资产库和黑板。
- 首页不展示复杂参数。
- Prompt 是首屏最高优先级。
- 热门 Skill 不超过两行，更多进入 Skill 页面。

核心状态：

- 未登录：可浏览首页公开作品和精选作品，可输入 Prompt；点击开始创作时弹登录弹窗。
- 无最近项目：展示创建新项目空态。
- Skill 加载失败：保留 Prompt，Skill 区展示错误重试。

## Agent 创作页模板

页面目标：

- 承载对话、模型选择、扣费确认、生成执行、A2UI、资产和黑板。

桌面结构：

```text
顶部 ProjectToolbar：当前项目 / 素材 / 项目 / 剪辑 / 导出 / 积分 / 用户
左侧 StoryboardPanel：故事板、关键元素、分镜、素材缩略图和黑板结构化内容
中央 PreviewStage：当前元素 / 分镜预览、模型、尺寸和生成结果
右侧 ChatPanel：对话消息、Thinking、Media Assets、确认、进度和输入框
```

规则：

- 普通用户和企业用户共用同一个 Agent 创作工作台。
- 工作台为沉浸创作模式，不展示全局 AppSideNav；通过顶部 ProjectToolbar 返回项目或资产入口。
- 当前空间切换等同重新登录对应身份，页面数据必须刷新。
- 模型选择入口在聊天输入框内。
- 资产和黑板是聊天外部组件：结构化内容优先进入 StoryboardPanel / PreviewStage，资产摘要可在 ChatPanel 内以 MediaAssetsCard 展示。
- 生成前扣费确认必须锁定模型和积分预估。

响应式：

- 小屏桌面右侧工作区改为 Drawer。
- 中间对话区保持最小可读宽度。
- 超宽屏可增加辅助预览，但不得无限拉宽对话行。

## 项目 / 会话模板

页面目标：

- 查看项目、会话历史和创作过程。
- 回到会话继续引用已生成资产创作。

结构：

```text
页面标题
筛选栏：类型 / 时间 / 状态 / 空间
视图切换：网格 / 列表
项目或会话列表
详情 Drawer
```

规则：

- 列表项显示封面、名称、类型、最后更新时间、状态。
- 企业空间普通成员只看自己的产物和会话。
- 个人身份只看个人空间数据。
- 支持按图片、音乐、视频、混合项目筛选。

状态：

- 空态：引导开始创作。
- 加载：Skeleton 卡片。
- 删除或归档：必须二次确认。

## 资产库模板

页面目标：

- 管理上传素材、生成资产和 Skill 中间产物。

结构：

```text
标题 + 上传按钮
筛选栏：类型 / 来源 / 状态 / 时间
资产网格或列表
资产详情 Drawer
```

规则：

- 资产类型包含图片、音乐、视频、文本、提示词、分镜、时间线等平台固定元素类型。
- 元素类型来自配置，不写死在前端。
- 支持预览、下载、引用到当前会话。
- 上传时显示格式、大小和失败规则。
- 保存失败资产不消失，展示重试或删除。

响应式：

- 桌面默认网格。
- 窄屏使用列表。
- Drawer 小屏全屏。

## Skill 页面模板

页面目标：

- 查看系统 Skill、个人 Skill、企业 Skill。
- 创建、编辑、提交审核和查看版本。

结构：

```text
Skill 分类 Tabs
筛选栏：来源 / 审核状态 / 类型
Skill 列表或网格
创建 Skill 入口
Skill 详情 Drawer
```

规则：

- 平台系统 Skill 对普通用户只读。
- 个人 Skill 由普通用户创建。
- 企业 Skill 只能企业拥有者创建。
- 企业 Skill 对企业成员全部可见。
- 发布前需要 3 个测试样例。
- 个人/企业 Skill 需要平台管理员审核。
- 系统 Skill 不需要审核。

状态：

- 草稿。
- 待审核。
- 已通过。
- 已拒绝。
- 已下线。
- 有新版本。

## Skill Builder 模板

页面目标：

- 完整创建 Skill，包括基础信息、内容、工具、测试样例和提交审核。

推荐结构：

```text
步骤导航
  1 基础信息
  2 Skill 内容
  3 可用 Tool
  4 测试样例
  5 预览与提交
右侧预览 / 校验结果
```

规则：

- 只能选择平台开放 Tool。
- Tool 风险等级展示为 Tag。
- 测试样例至少 3 个。
- 保存草稿和提交审核分开。
- 审核意见不必填。

## 积分中心模板

页面目标：

- 展示积分余额、即将过期积分、消耗明细、兑换码入口。

结构：

```text
余额概览
即将过期积分
兑换码输入
流水筛选
积分流水列表
```

规则：

- 第一版积分由后台手动发放和兑换码兑换。
- 积分不永久有效，必须显示有效期。
- 生成任务使用冻结积分流程。
- 企业空间展示企业积分，个人空间展示个人积分。
- 普通企业成员可看自己的消耗明细。

状态：

- 兑换成功。
- 兑换码无效。
- 兑换码已使用。
- 兑换码不属于当前用户或企业。
- 积分不足。

## 个人作品中心模板

页面目标：

- 管理当前用户自己的作品集。

结构：

```text
标题 + 创建/导入入口
筛选栏：类型 / 分类 / 公开状态 / 时间
作品网格
作品详情 Drawer 或详情页
```

规则：

- 展示个人作品，不展示企业其他成员作品。
- 作品支持公开分享。
- 公开后可进入精选作品展示范围。
- 支持编辑标题、简介、标签、分类、封面。
- 删除和取消公开必须确认。

## 精选作品中心模板

页面目标：

- 展示分享后的公开作品，支持免登录查看。

结构：

```text
顶部公开导航
分类 Tabs
标签筛选
作品类型筛选
响应式作品网格
```

规则：

- 免登录可查看。
- 支持点赞、分享、作品标签、作品分类。
- 未登录点赞时弹出登录弹窗并保留当前作品上下文。
- 不展示源会话、黑板、提示词、积分、模型成本和隐私信息。
- 详情页展示公开媒体、简介、作者公开名、标签、分类和相关推荐。

## 个人中心模板

页面目标：

- 管理账号资料、登录身份、主题偏好、通知和空间切换。

结构：

```text
二级导航
资料设置
身份与空间
通知
主题
安全
```

规则：

- 个人/企业身份切换在个人中心触发。
- 切换身份等同重新登录对应身份。
- 登录页面可选择企业登录和个人登录。
- 企业用户被移除企业后自动变为普通用户。

## 企业空间模板

页面目标：

- 企业拥有者管理企业成员和企业积分。

结构：

```text
企业信息
成员列表
邀请成员
转让拥有者
企业积分摘要
```

规则：

- 企业管理员暂时等同企业拥有者。
- 企业成员不能主动退出。
- 拥有者可在企业成员之间转让。
- 转让后原拥有者变为普通企业成员。
- 企业空间不做复杂业务管理云。

## 通知中心模板

页面目标：

- 接收 Skill 审核通知、生成完成、失败、兑换结果等站内信。

结构：

```text
通知分类
通知列表
通知详情
批量已读
```

规则：

- Skill 审核结果通过站内信触达。
- 通知列表显示类型、标题、时间和已读状态。
- 与扣费、安全、审核相关的通知必须可追溯。

## 错误页模板

| 页面 | 场景 | 主操作 |
| --- | --- | --- |
| 403 | 无权限或空间不匹配 | 返回工作台 / 切换身份 |
| 404 | 页面不存在 | 返回首页 |
| 500 | 服务异常 | 重试 / 联系支持 |
| Offline | 网络中断 | 重试 |

规则：

- 错误页不展示技术堆栈。
- 必须有恢复路径。
- 企业空间权限问题必须提示当前身份和空间。

## 页面状态规范

每个页面必须覆盖：

- loading。
- empty。
- error。
- success。
- permission denied。
- offline 或 reconnecting。

Agent 创作页额外覆盖：

- streaming。
- thinking。
- confirming。
- generating。
- interrupted。
- resume。
- save_failed。

## 响应式验收

必须截图验收：

- `1440px` 标准桌面。
- `1920px` 大屏。
- `2560px` 宽屏。
- `768px` 窄桌面或平板宽度。

验收规则：

- 左侧导航、ContextHeader、主内容不重叠。
- 文字不溢出按钮或卡片。
- hover 不造成布局跳动。
- Drawer 和 Modal 在小屏可用。
- 表格和列表有横向滚动或字段降级策略。

## 前端落地要求

- 页面使用统一 App Shell，不为每个页面重新实现导航和 ContextHeader。
- 页面模板组件化，不复制粘贴布局。
- 页面状态组件复用公共规范。
- 主题切换只通过 token，不写页面级硬编码颜色。
- C 端页面不得使用平台后台组件风格。

## 验收标准

- 所有 C 端核心页面都能归入本规范的页面模板。
- 每个页面都有明确主任务和首屏优先级。
- 未登录、个人空间、企业空间的页面入口和数据范围清楚。
- 页面 loading、empty、error、permission denied 状态完整。
- 页面在不同显示器下保持弹性布局和可读性。
