## 目标用户

- 个人用户：快速开始个人空间创作。
- 企业成员 / 企业拥有者：在企业身份下快速开始企业空间创作。
- 未登录访客：浏览公开作品和公开能力，触发创作时登录。

## 主任务

让未登录访客可以浏览公开内容；让登录用户确认当前空间后，从一个 Prompt、素材或高频场景创建项目，并进入统一 Agent 工作台。

## 页面区域

| 区域 | 作用 |
| --- | --- |
| 左侧导航 | 未登录展示公开入口、登录和注册；登录后展示新建创作、工作台、项目、资产、Skill、作品、精选和积分入口。 |
| 右侧 ContextHeader | 登录后展示当前空间、积分、通知和用户菜单；未登录展示登录 / 注册入口。 |
| 右侧创作入口 | 承载 Prompt、生成类型入口、素材入口和开始创作动作。 |
| 快捷开始 | 只展示少量高频场景，不展示完整 Skill 矩阵。 |
| 公开作品 | 未登录和登录状态都可展示公开作品摘要。 |
| 最近项目 | 登录后展示最近项目，空态引导创建第一个项目。 |
| 精选作品露出 | 引导进入公开精选作品，不在首页完整承载作品广场。 |

## 关键组件

| 组件 | 用途 |
| --- | --- |
| AppShell | 登录后 C 端左右结构，包含 AppSideNav、ContextHeader 和 PageBody。 |
| PublicSideNav | 未登录公开首页的左侧轻导航。 |
| SpaceBadge | 展示个人空间或企业空间，事实以后端身份上下文为准。 |
| PromptComposerLite | 首页轻量 Prompt，不承载完整模型参数和扣费确认。 |
| QuickSkillCard | MV、音乐、商品图等高频入口。 |
| RecentProjectStrip | 最近项目或空态入口。 |
| FeaturedWorkStrip | 精选作品预览和跳转。 |
| LoginModal | 未登录触发创作、上传、选择资产、点赞或个人入口时弹出。 |

## 状态覆盖

| 状态 | 首页表现 |
| --- | --- |
| loading | Prompt 保持可见，最近项目和 Skill 使用 Skeleton。 |
| empty | 未登录无公开作品时展示精选作品入口；登录后无最近项目时展示开始创作入口。 |
| error | Skill 或最近项目加载失败时局部展示重试，不阻断 Prompt。 |
| login_required | 未登录点击开始创作、上传素材、选择资产或点赞时弹出登录弹窗。 |
| success | 登录后点击开始创作，创建项目并进入工作台，保留当前输入和素材上下文。 |
| streaming | 首页不直接消费；跳转工作台后展示。 |
| interrupt | 首页不直接承接；工作台确认面板承接。 |
| resume | 首页不直接恢复 run；最近项目可进入会话恢复页。 |

## AG-UI / API / RPC 依赖

| 依赖 | 说明 |
| --- | --- |
| Public API | 首页公开作品、公开 Skill 摘要、精选作品摘要。 |
| API | 登录后当前身份、当前空间、积分摘要、最近项目、创建项目。 |
| AG-UI | 首页不直接消费，进入 `/app/workspace` 后由工作台消费。 |
| RPC | 首页不直接调用业务 RPC；业务能力通过 API 或后续契约承接。 |

## 待确认项

- 未登录用户输入 Prompt 后是否登录即继续创建项目，默认建议保留 Prompt 并继续。
- 首页展示的热门 Skill 是否由运营配置、系统 Skill 排序或固定推荐生成。
- 项目标题是否由第一条 Prompt 自动生成，是否允许首页直接编辑。
- 最近项目卡片展示哪些摘要字段，需 API 契约确认。

## 边界声明

本草图不定义后端字段。公开作品、当前空间、Skill、积分、最近项目、精选作品字段全部以后续 Public API / API / RPC / AG-UI 契约为准。
