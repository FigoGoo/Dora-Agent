# 企业空间成员积分 Skill 概览草图

状态：draft
owner：产品体验设计师
更新时间：2026-06-25
适用范围：企业空间概览、企业成员、企业积分、企业 Skill 和本人企业消耗页面低保真草图
相关代码路径：用户端 `frontend/**`；管理端 `admin_frontend/**`
相关契约：[C 端与后台业务 API 契约草案](../../contracts/api/C端与后台业务API契约草案.md)、[业务微服务 RPC 契约草案](../../contracts/rpc/业务微服务RPC契约草案.md)；企业管理页默认不直接消费 AG-UI

## 目标用户

- 企业拥有者：邀请成员、移除成员、转让拥有者、查看企业积分和管理企业 Skill。
- 企业成员：进入企业空间创作，查看本人企业空间消耗。

## 主任务

让企业拥有者快速理解企业空间状态，完成成员、积分和企业 Skill 管理，同时让企业成员明确自己没有成员管理和企业总账权限。

│ + 新建创作 │ 企业二级导航   │ 企业概览                                      │
│ 首页       │ 概览          │ ┌─────────────┐ ┌─────────────┐ ┌───────────┐ │

## 页面区域

| 区域 | 作用 |
| --- | --- |
| AppSideNav | 企业身份沿用用户端左侧主导航，通过当前空间区分个人和企业。 |
| 企业空间标识 | 右侧 ContextHeader 明确当前企业名称和企业空间，避免误用企业积分。 |
| 企业二级导航 | 概览、成员、企业 Skill、企业积分、本人消耗。 |
| 摘要卡 | 展示企业身份、积分摘要和企业 Skill 摘要。 |
| 成员模块 | 邀请、邀请状态、移除、拥有者转让入口。 |
| 企业 Skill 模块 | 审核状态、创建入口、详情入口。 |
| 企业积分模块 | 余额、即将过期、兑换码、流水和成员消耗。 |

## 关键组件

| 组件 | 用途 |
| --- | --- |
| EnterpriseShell | 企业空间二级导航和权限状态。 |
| EnterpriseSummary | 企业摘要和当前角色。 |
| MemberTable / InviteDialog | 成员列表、邀请和邀请状态。 |
| RemoveConfirm / OwnerTransferConfirm | 移除成员和拥有者转让确认。 |
| EnterpriseSkillList | 企业 Skill 状态和创建入口。 |
| CreditOverview / RedeemForm / MemberUsageList | 企业积分、兑换和成员消耗。 |
| PermissionState | 企业成员访问拥有者页面时展示无权限。 |

## 状态覆盖

| 状态 | 企业空间表现 |
| --- | --- |
| loading | 企业身份、成员、Skill、积分列表加载时显示 Skeleton。 |
| empty | 无成员邀请、无企业 Skill、无流水时给出对应主操作或说明。 |
| error | 邀请、移除、转让、兑换或列表加载失败时展示可重试错误。 |
| success | 邀请发送、移除成功、转让成功、兑换成功后刷新对应模块。 |
| streaming | 企业管理页不直接消费；企业身份进入工作台后消费。 |
| interrupt | 企业管理页的危险动作使用 Modal 确认；工作台扣费确认由 AG-UI 承接。 |
| resume | 企业管理页不恢复 run；企业会话恢复在工作台处理。 |

## AG-UI / API / RPC 依赖

| 依赖 | 说明 |
| --- | --- |
| API | 当前企业身份、企业摘要、成员列表、邀请状态、企业 Skill、企业积分和本人消耗。 |
| RPC | 企业邀请、移除成员、拥有者转让、企业积分兑换、企业 Skill 审核流转等业务能力。 |
| AG-UI | 企业管理页默认不消费；企业身份下的 `/app/workspace` 消费同一工作台事件。 |

## 权限边界

| 角色 | 可见内容 |
| --- | --- |
| 企业拥有者 | 成员、企业 Skill、企业积分、成员消耗汇总、本人企业消耗。 |
| 企业成员 | 企业工作台入口、本人企业消耗、本人企业空间资产和作品。 |
| 非企业身份 | 企业页面展示无权限或引导切换身份。 |

企业拥有者不额外查看成员资产、会话、任务、黑板、生成记录或成员私有作品。

## 边界声明

本草图只表达企业空间页面结构，不定义企业表字段、权限字段或积分字段。企业事实由业务微服务维护，Agent 微服务只在工作台中接收当前空间上下文。
