## 目标用户

- 个人用户：管理自己的创作项目、继续创作、查看资产和创建作品。
- 企业成员 / 企业拥有者：在企业空间内查看本人创建的企业空间项目。

## 主任务

让用户清楚知道一次创作产生的会话、资产、黑板和作品属于哪个项目，并能从项目继续创作。

## 页面区域

| 区域 | 作用 |
| --- | --- |
| 窄图标导航 | 项目列表页使用沉浸式窄导航，当前高亮项目。 |
| 顶部项目栏 | 展示产品名、当前页面、积分和用户状态。 |
| 项目列表工具条 | 搜索、归档筛选、新建项目。 |
| 项目卡片 | 展示项目封面、标题、会话数量、资产数量和最近更新时间。 |
| 项目详情头部 | 展示项目标题、空间、状态、继续创作和归档入口。 |
| 项目标签页 | 聚合概览、会话、资产、黑板和作品。 |
| 工作台项目栏 | 确认当前对话和资产写入哪个项目。 |

## 关键组件

| 组件 | 用途 |
| --- | --- |
| ProjectCard | 项目列表卡片。 |
| ProjectHeader | 项目详情头部。 |
| ProjectTabs | 项目详情标签页。 |
| SessionList | 项目下会话和恢复入口。 |
| AssetGrid | 项目资产网格和资产操作。 |
| BlackboardPreview | 项目最新黑板和快照摘要。 |
| WorkStrip | 项目关联作品。 |

## 状态覆盖

| 状态 | 表现 |
| --- | --- |
| loading | 项目卡片、详情头部、标签内容使用 Skeleton。 |
| empty | 无项目时引导开始创作；项目无资产时引导继续创作或上传素材。 |
| filtered_empty | 项目筛选无结果时支持清除筛选。 |
| error | 列表或详情加载失败时提供重试。 |
| permission_denied | 用户访问旧空间或无权限项目时不展示项目内容。 |
| resume | 从项目会话恢复时进入工作台并显示恢复条。 |

## 契约依赖

| 依赖 | 说明 |
| --- | --- |
| API | 项目列表、项目详情、归档、重命名、项目资产、项目会话和项目作品。 |
| RPC | 项目权限、资产归属、作品创建和归档写操作。 |
| AG-UI | 工作台流式事件建议携带 `project_id`，用于当前项目资产和黑板刷新。 |

## 待确认项

- 项目封面默认取第一张可用资产还是用户手动选择。

## 边界声明

项目是业务事实，Agent 只持有项目上下文。项目详情不展示源提示词、系统提示词、模型成本、积分明细正文、供应商响应或平台内部策略。
