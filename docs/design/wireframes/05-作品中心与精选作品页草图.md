## 目标用户

- 登录用户：整理自己的作品、编辑信息、分享或取消分享。
- 企业成员：在企业身份下管理本人创建的企业空间作品。
- 未登录访客：免登录浏览首页公开作品、精选作品列表和详情。
- 平台管理员：通过后台下架风险公开作品。

## 主任务

把已保存资产整理成个人作品，并通过公开快照进入精选作品中心；公开展示时保护源会话、黑板、提示词、积分和隐私边界。

## 页面区域

| 区域 | 作用 |
| --- | --- |
| AppSideNav / PublicSideNav | 个人作品中心使用登录后左侧导航；精选作品使用未登录可访问的左侧公开导航。 |
| 个人作品 FilterBar | 按资源类型、分类、标签、分享状态和时间筛选。 |
| 作品网格 | 展示当前登录用户自己的作品，不展示他人成员作品。 |
| 作品编辑 Drawer | 从已保存资产创建作品，编辑公开展示信息。 |
| 分享确认 | 生成公开快照前确认公开影响并触发文本安全评估。 |
| 精选作品公开 Shell | 未登录也可浏览公开作品。 |
| 公开作品详情 | 展示公开媒体、标题、简介、标签、作者公开昵称、点赞和分享。 |

## 关键组件

| 组件 | 用途 |
| --- | --- |
| WorkGalleryGrid | 个人作品网格。 |
| WorkCard | 作品封面、标题、分类、标签、状态和操作。 |
| AssetSelector | 从有权限的已保存资产选择作品来源。 |
| WorkForm | 标题、封面、简介、分类和标签编辑。 |
| ShareModal | 分享影响说明、公开快照、安全评估和确认。 |
| PublicShell | 免登录公开页面壳。 |
| PublicWorkGrid / PublicDetail | 精选作品列表和详情。 |
| LikeButton / ShareButton | 登录后点赞、未登录弹出登录弹窗并保留当前作品上下文、复制公开链接。 |

## 状态覆盖

| 状态 | 作品页表现 |
| --- | --- |
| loading | 作品网格、公开列表、详情和编辑 Drawer 使用 Skeleton。 |
| empty | 无个人作品、无筛选结果、无公开作品时展示创建或清除筛选入口。 |
| error | 列表、保存、分享、点赞失败时展示可理解错误和重试。 |
| success | 保存作品、分享、取消分享、点赞、复制链接成功后反馈。 |
| streaming | 作品页不消费；作品来源资产由工作台生成时消费 AG-UI。 |
| interrupt | 分享前确认用 Modal；不使用 AG-UI interrupt。 |
| resume | 作品页不恢复 run；可从作品关联来源进入原会话恢复。 |

## AG-UI / API / RPC 依赖

| 依赖 | 说明 |
| --- | --- |
| API | 作品列表、作品详情、创建、编辑、分享、取消分享、公开列表、公开详情、公开快照 TOS 公共 URL、点赞、复制链接。 |
| RPC | 作品业务事实、公开快照、分享状态、点赞去重、公开作品下架等业务能力。 |
| 内容安全 | 分享前评估标题、简介、标签等公开文本。 |
| AG-UI | 作品页不直接消费；生成资产过程在工作台消费。 |

## 公开展示边界

公开作品只展示：

- 标题、封面、简介、公开媒体、分类、标签、作者公开昵称、点赞数和分享入口。
- 公开媒体来自公开快照媒体前缀下的 TOS 公共 URL，不直接返回源业务资产 object key。

公开作品不展示：

- 源会话、黑板、提示词、私有素材、积分、模型成本、内部模型信息、用户邮箱、手机号或内部用户 ID。

## 待确认项

- 作品分类和标签来源是系统字典、运营配置还是契约枚举。
- TOS 已接入 CDN，域名为 `https://tos.doraigc.com`；第一版不做 CDN 缓存失效。
- TakenDown 后创建者是否可编辑后重新分享，需要业务流程确认。
- 未登录访客复制链接是否需要限制频次或风控提示。

## 边界声明

本草图不定义作品、资产、点赞、公开快照或公开媒体字段。作品不是资产本身，作品引用一个或多个已保存资产；资产保存失败不能创建作品。
