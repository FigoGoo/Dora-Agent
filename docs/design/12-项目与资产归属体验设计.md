## 背景

统一 Agent 工作台会持续生成资产和过程信息。用户需要一个比“会话”更稳定的容器来理解“我这次创作的东西在哪里”。项目就是这个容器：它承接创作目标，聚合会话、资产、黑板和作品。

## 体验目标

- 让用户从首页开始创作时自然形成项目，不额外增加阻力。
- 让工作台始终展示当前项目，避免资产归属不清。
- 让项目详情成为继续创作、查看过程、管理资产和创建作品的聚合入口。
- 让资产详情能说明所属项目和来源会话。
- 让作品创建从项目资产出发，减少跨资产库挑选的混乱。

## 信息架构

```text
/app
  -> 最近项目
  -> 新建项目并进入 /app/workspace?projectId=...

/app/projects
  -> 项目列表

/app/projects/:projectId
  -> 项目概览
  -> 会话
  -> 资产
  -> 黑板
  -> 作品

/app/workspace?projectId=...
  -> 当前项目下的新会话或恢复会话
```

最终路由由前端开发工程师结合框架确认。草案阶段只表达页面职责和归属关系。

## 项目列表

| 区域 | 作用 |
| --- | --- |
| ContextHeader 和工具条 | 当前空间、项目搜索、归档筛选、新建项目。 |
| 项目网格 | 展示项目标题、封面、最近资产、最近更新时间、会话数量和作品数量。 |
| 空状态 | 引导从 Prompt 创建第一个项目。 |
| 归档视图 | 展示已归档项目，不默认混入最近项目。 |

状态覆盖：

- loading：项目卡片骨架。
- empty：展示开始创作入口。
- filtered_empty：支持清除筛选。
- error：局部重试，不影响顶部新建入口。
- permission_denied：切换身份后访问旧空间项目时展示返回入口。

## 项目详情

项目详情采用标签页，不把所有内容堆在一屏。

| 标签 | 内容 | 关键组件 |
| --- | --- | --- |
| 概览 | 项目标题、封面、最近会话、最近资产、继续创作 | ProjectHeader、ContinueButton、RecentActivity |
| 会话 | 项目下的会话列表，支持恢复和新建会话 | SessionList、SessionCard、RestoreAction |
| 资产 | 项目资产网格，支持预览、下载、引用、创建作品 | AssetGrid、AssetDrawer、AssetActionBar |
| 黑板 | 项目最新黑板和 run 快照 | BlackboardPreview、SnapshotTimeline |
| 作品 | 由项目资产创建的作品 | WorkStrip、CreateWorkAction |

项目详情不展示模型成本、积分明细正文、系统提示词、供应商响应或内部策略。

## 工作台项目上下文

工作台右侧 ContextHeader 需要展示当前项目：

- 项目标题和当前空间。
- 返回项目详情入口。
- 新建会话入口。
- 项目状态，例如 active / archived。

工作台左侧会话栏展示当前项目内的会话，而不是全局会话。资产和黑板面板展示当前项目资产、当前会话产物和项目黑板最新视图。

## 资产归属展示

资产卡片和资产详情需要展示：

- 所属项目摘要。
- 来源类型：生成、上传、导入。
- 来源会话摘要或 run 摘要。
- 当前状态：uploading、available、save_failed。
- 可操作动作：预览、下载、引用到当前会话、创建作品。

字段名、来源摘要和可操作动作以后续 API / RPC 契约为准。

## 作品创建入口

从项目创建作品时：

- 默认筛选当前项目可用资产。
- 用户选择封面和作品资产。
- 公开分享前走内容安全和公开快照规则。
- 创建成功后作品回写到项目详情的“作品”标签。

## 契约依赖

| 依赖 | 页面影响 |
| --- | --- |
| Project API | 项目列表、详情、重命名、归档、新建项目。 |
| Agent API | 创建会话和 run 时传入或解析项目上下文。 |
| Asset API / RPC | 项目资产列表、来源摘要、引用权限、创建作品选择器。 |
| AG-UI | 工作台事件建议带 `project_id`，用于刷新当前项目资产和黑板。 |

## 待确认事项

- 项目列表是否展示会话数量和作品数量。
- 项目详情是否保留单独黑板标签，还是在概览中展示黑板摘要。
- 归档项目是否允许继续创作，建议第一版归档后需要先恢复。
- 全局资产库是否默认展示所有项目资产，还是默认按最近项目聚合。

## 验收标准

- [ ] 首页开始创作能解释会创建项目。
- [ ] 项目列表和项目详情覆盖 loading、empty、filtered_empty、error、permission_denied。
- [ ] 工作台 ContextHeader 能展示当前项目。
- [ ] 项目详情能聚合会话、资产、黑板和作品。
- [ ] 资产详情能展示所属项目和来源会话 / run 摘要。
- [ ] 作品创建能从项目资产选择。
- [ ] 企业空间第一版不增加成员项目可见性。
