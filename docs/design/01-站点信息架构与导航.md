## 导航设计目标

- 参考 Flova 的创作工具结构，将项目、生成、资产、Skill、探索内容作为一级心智。
- 让用户始终知道当前身份和当前空间。
- 让 Agent 创作入口保持最高优先级。
- 让用户端桌面 Web 采用稳定的左右结构：左侧导航，右侧内容。
- 让个人作品中心和精选作品中心区分清楚：前者是自己的作品集，后者是公开展示区。
- 让未登录访客可以浏览与用户无关的公开内容，需要登录的动作由登录弹窗承接。
- 平台后台采用独立入口和独立代码，不与用户端混用。

## 参考结构映射

Dora-Agent 可以借鉴 Flova 的一级结构组织方式，但导航命名、页面职责和数据权限以本项目 PRD 为准。

| 参考结构 | Dora-Agent 一级入口 | 页面职责 |
| --- | --- | --- |
| Home | 工作台首页 / 公开首页 | 公开作品、快捷创作、登录后最近项目和账户空间概览 |
| Projects | 项目 / 会话 | 创作项目、会话历史、创作过程回看 |
| Generator | Agent 创作 | 统一 Agent 输入、模型选择、生成确认和创作执行 |
| Asset Library | 资产库 | 上传素材、生成素材、中间素材、可引用资产 |
| Skill | Skill | 系统 Skill、个人 Skill、企业 Skill 和 Skill Builder |
| Explore / FlovaTV | 精选作品 | 分享后的公开作品浏览、点赞、分享、分类和标签 |

结构借鉴边界：

- 可以采用“工作台首页 + 项目 + 快速创作 + 资产库 + Skill + 精选作品”的一级心智。
- 不复制 Flova 的品牌、Logo、图片、商业文案、具体页面排版和交互动效。
- Dora-Agent 必须额外强化当前空间、积分、扣费确认、黑板、A2UI 和企业身份。

## 用户端左右结构

桌面 Web 用户端统一采用左右结构：

```text
左侧 AppSideNav
  Logo / Dora-Agent
  新建创作
  首页
  工作台
  项目 / 会话
  资产库
  Skill
  作品中心
  精选作品
  积分
  底部：主题 / 登录或用户入口

右侧 PageContent
  ContextHeader：当前页面标题 / 当前空间 / 积分 / 通知 / 用户
  PageBody：当前页面主体
```

结构规则：

- 主导航不再放在顶部横向展开。
- 左侧 AppSideNav 承载一级入口和高频创建动作。
- 右侧 ContextHeader 承载当前空间、积分、主题、通知和用户菜单。
- 登录后页面和未登录公开页面都使用左右结构；未登录状态下左侧只展示公开入口、登录和注册。
- Agent 创作工作台进入沉浸创作模式时可隐藏全局 AppSideNav，改用顶部项目工具条 + 左侧大画布 + 右侧对话面板，最大化创作画布。
- 平台后台仍使用独立后台壳，不与用户端 AppSideNav 混用。

当前空间展示：

- 个人空间：显示“个人”标识。
- 企业空间：显示企业名称和“企业空间”标识。
- 企业空间必须有明显视觉提示，避免用户误用企业积分生成个人内容。

## 左侧导航结构

桌面 Web 左侧导航：

| 区域 | 内容 |
| --- | --- |
| 品牌区 | Logo、产品名、当前版本或环境提示 |
| 主创建入口 | 新建创作，优先进入统一 Agent 工作台 |
| 一级导航 | 首页、工作台、项目、资产库、Skill、作品中心、精选作品、积分 |
| 上下文入口 | 当前空间切换、个人中心、企业管理入口 |
| 底部工具 | 主题切换、帮助、登录 / 用户菜单 |

左侧导航规则：

- 小屏桌面可折叠为图标栏。
- 超宽屏可保持展开并显示二级信息。
- 侧边栏不承载复杂表单。
- 当前页面高亮只表达路由位置，不表达业务权限。
- 企业空间不更换导航结构，只切换当前空间上下文和可见入口。

## 公开页面导航

精选作品中心支持免登录查看。

首页公开作品也支持免登录查看，但最近项目、资产、积分、通知和项目详情属于登录后数据。

| 状态 | 导航 |
| --- | --- |
| 未登录 | 左侧：Logo、首页、精选作品、公开 Skill；底部：登录、注册 |
| 已登录 | 左侧：Logo、新建创作、首页、工作台、项目、资产库、Skill、作品中心、精选作品、积分；右侧 ContextHeader 展示空间、通知和用户菜单 |

未登录访客点击开始创作、点赞、项目、资产库、作品中心、积分或通知入口时，前端弹出登录弹窗并保留当前意图。

精选作品中心页面结构：

- 顶部分类筛选。
- 标签筛选。
- 资源类型筛选：图片、音乐、视频、混合。
- 作品网格。
- 详情页展示公开媒体、简介、标签、分类、点赞、分享。

## 企业空间导航

企业空间不做独立复杂工作台。进入企业身份后，用户端导航基本一致，但数据上下文切换为企业空间。

企业拥有者额外可见：

- 企业成员管理。
- 企业积分。
- 企业 Skill。

企业普通成员可见：

- 企业工作台。
- 企业空间本人作品和资产。
- 本人企业空间积分消耗明细。

## 平台后台导航

平台后台独立入口，推荐左侧导航 + 顶部账号栏 + 主内容区。

| 后台模块 | 页面 |
| --- | --- |
| 概览 | 后台首页 |
| 账号与用户 | 平台管理员账号、用户管理 |
| Skill | 系统 Skill 管理、Skill 审核队列 |
| 模型 | 模型供应商管理、模型管理 |
| Tool | Tool 管理 |
| 积分 | 积分发放、兑换码管理 |
| 内容展示 | 精选作品管理 |
| 系统 | 平台审计日志 |

后台规则：

- 不进入普通用户 Agent 工作台。
- 不代登录普通用户。
- 不展示用户私有创作内容。
- 表格页必须支持筛选、分页、空态、错误态。

## 响应式导航

| 宽度 | 导航策略 |
| --- | --- |
| 小于 768px | 左侧导航收进 Drawer；只保证浏览、查看、基础对话 |
| 768-1199px | 左侧 AppSideNav 折叠为图标栏，右侧内容优先单面板 |
| 1200-1599px | 标准左侧 AppSideNav + 右侧内容；工作台内部可展示 Agent 主区和 Workspace |
| 1600px 以上 | 左侧 AppSideNav 固定，右侧可展示多辅助面板，内容设置最大阅读宽度 |
