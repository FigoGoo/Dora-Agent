## 设计目标

- 参考 Flova 的“快速生成 + 项目/资产/Skill”结构，形成 Dora-Agent 的核心创作空间。
- 让用户从一个输入框开始创作，同时能看到资产和黑板。
- 支持视频、MV、短视频等复杂创作的分镜、脚本、提示词和素材组织。
- 支持不同显示器下的弹性面板。
- 工作台参考沉浸式创作室结构：顶部项目工具条，左侧故事板，中间预览区，右侧对话面板。

## 桌面布局

推荐默认结构：

```text
顶部项目工具条：Dora-Agent / 项目标题 / 素材 / 项目 / 剪辑 / 复制 / 导出 / 积分 / 用户
主工作区
  左侧 StoryboardPanel：故事板、关键元素、分镜、素材缩略图
  中央 PreviewStage：当前元素 / 分镜预览、模型和尺寸信息、生成结果
  右侧 ChatPanel：对话、消息流、Media Assets、结果表格、输入框、模型、Skill、资产库入口
```

## 面板比例

| 显示环境 | StoryboardPanel | PreviewStage | ChatPanel |
| --- | --- | --- | --- |
| 笔记本小屏 | `28-32%`，可折叠为抽屉 | `34-38%`，保留预览主体 | `30-36%`，输入区固定底部 |
| 标准桌面 | `30-34%` | `34-40%` | `28-32%` |
| 宽屏 | 固定舒适扫描宽度 | 居中扩大预览，不无限拉伸图片 | 固定舒适阅读宽度 |
| 超宽屏 | 不超过故事板最大宽度 | 保持预览最大内容宽度 | 不把消息无限拉宽 |

第一版可以先做预设比例，后续支持拖拽调宽。若实现拖拽，必须提供最小宽度和一键恢复默认。

## Agent 对话区

对话区组件：

- 用户消息。
- Agent 文本回复。
- 思考状态打印机。
- Skill / Tool / Model / Status Tag。
- 积分预估卡片。
- 人工确认卡片。
- 生成进度卡片。
- 错误提示卡片。

设计规则：

- 对话流密度中等，不做过大聊天气泡。
- 思考状态默认轻量展开，完成后可折叠。
- Tag 必须短小、可扫描，不占用过多垂直空间。
- 进度卡片展示当前状态、可取消动作、已完成产物摘要。

## 输入框设计

输入框参考“由一个想法或故事开始”的创作入口心智，但 Dora-Agent 需要承载更多控件。

输入框包含：

- 主文本输入。
- 上传 / 选择素材。
- 模型选择入口：图片、音乐、视频。
- 单选、多选、参数输入。
- 发送按钮。

模型选择规则：

- 不做独立 A2UI 组件。
- 在输入框内以 Chip / Popover 呈现。
- 用户不选择时显示默认模型。
- 进入扣费确认后锁定。

## 故事板与预览区

聊天外部工作区由 StoryboardPanel 和 PreviewStage 组成：

| 区域 | 用途 |
| --- | --- |
| StoryboardPanel | 展示 Skill 输出元素、分镜、脚本、提示词、素材缩略图和生成状态。 |
| PreviewStage | 展示当前选中元素 / 分镜的生成预览、模型、尺寸、切换控制和空态。 |
| ChatPanel 内 Media Assets | 展示本轮完成的资产摘要、结果表格和下一步提示。 |

默认打开规则：

- 图片、音乐普通生成：默认选中最近生成资产并在 PreviewStage 预览。
- 视频、MV、短视频、广告视频：默认打开 StoryboardPanel 的故事板或关键元素分组。
- 用户手动切换后当前会话保持上次选择。

## 黑板设计

黑板用于结构化创作，不是普通富文本笔记。

视频类黑板结构：

- 故事线摘要。
- 分镜列表。
- 每个分镜包含画面、脚本、提示词、素材引用、生成状态。
- 时间线摘要。
- 当前生成状态。

资产元素渲染：

- 短文本、长文本、富文本。
- 图片、音频、视频引用。
- 提示词。
- 分镜。
- 时间线。
- 标签组、参数组。

## 资产区设计

资产卡片必须支持：

- 图片缩略图。
- 音频播放状态。
- 视频封面和时长。
- 资源类型 Tag。
- 保存中、保存失败、可用状态。
- 预览、下载、引用、加入作品中心。

## 状态设计

| 状态 | UI 表现 |
| --- | --- |
| loading | 骨架和轻量进度 |
| streaming | 文本增量、停止按钮 |
| thinking | 打印机状态，可折叠 |
| confirming | 明确影响、积分和确认按钮 |
| generating | 进度、当前步骤、取消入口 |
| partial_completed | 展示已完成产物和未完成状态 |
| blocked | 安全阻断，提示修改 |
| reconnecting | 保留当前内容，显示恢复状态 |
| save_failed | 保留过程，提示保存失败 |

## 弹性布局约束

- StoryboardPanel 承载故事板、关键元素、分镜、素材缩略图和黑板结构化内容，列表独立滚动。
- PreviewStage 承载生成预览、空态引导、当前元素切换和模型 / 尺寸信息。
- ChatPanel 承载对话消息、Media Assets 结果卡和输入控件，输入区固定在面板底部。
- 工作台沉浸模式不展示全局 AppSideNav，项目列表、资产库等入口通过顶部项目工具条或返回项目页进入。
- ChatPanel 必须保持最小可读宽度。
- 卡片和按钮不能因长文件名、长模型名、长 Tag 溢出。
- 超宽屏不把对话气泡无限拉宽。
