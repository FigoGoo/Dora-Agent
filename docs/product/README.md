## 后续产品变更规则

- 新增或调整产品范围时，先在 `iterations/` 新建产品迭代文档。
- 影响核心流程、权限、计费、安全、契约或验收标准时，同步更新对应 PRD。
- 已归档的前置产品系统设计只用于追溯，不承接新需求。
