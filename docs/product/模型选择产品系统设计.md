## 背景

产品已确认：用户在使用生成资源类型模型时，可以在对话框指定模型。用户选择的模型只对当前对话生效，不保存为长期偏好。

## 目标

- 支持普通用户为图片、音乐、视频生成选择本次生效模型。
- 用户未选择模型时使用平台默认模型。
- 只向普通用户展示模型名称。
- 不允许普通用户选择文本模型或接入自己的模型 API Key。
- 让模型选择与积分预估、生成确认流程协同。

## 可选择模型范围

| 生成类型 | 是否可选模型 | 说明 |
| --- | --- | --- |
| 图片生成 | 是 | 文生图、图生图、风格化 |
| 音乐生成 | 是 | 文生音乐、歌词生成歌曲、BGM |
| 视频生成 | 是 | 文生视频、图生视频、分镜转视频 |
| 文本模型 | 否 | 平台默认，用于对话、意图识别、Skill 路由、文案和脚本 |
| 视觉理解模型 | 否 | 由平台和 Tool 决定 |
| 语音模型 | 后续 | TTS、ASR 后续再做 |

## 用户流程

1. 用户在统一 Agent 对话框输入生成意图。
2. Agent 识别为图片、音乐或视频生成任务。
3. 前端在用户聊天输入框内按图片、音乐、视频资源类型分组展示可用模型名称。
4. 用户选择当前对话使用的模型，也可以使用系统当前默认模型。
5. 用户补充生成参数，例如数量、时长、风格、参考图或歌词。
6. 系统对用户提示词和生成前提示词做 LLM 安全评估。
7. 安全评估通过后，系统按本次选择模型的积分单价进入积分预估。
8. 进入扣费确认后，不允许再修改模型。
9. 用户确认后，Agent 使用当前对话选择的模型执行生成。
10. 当前对话结束后，该模型选择失效，不保存为长期偏好。

## 业务规则

- 普通用户只能看到模型名称。
- 普通用户不能看到模型供应商、API Key、endpoint、内部模型 ID、内部成本和调用参数。
- 用户选择的模型只对当前对话生效。
- 用户不选择模型时，使用平台默认模型。
- 进入扣费确认后，不允许修改模型。
- 模型列表按图片、音乐、视频资源类型分组展示。
- 模型选择在聊天输入框内完成，不作为独立 A2UI 组件。
- 用户选择不同模型时，积分预估可能不同。
- 提示词安全评估通过后，才进入积分预估。
- 用户不能选择文本模型。
- 后台不配置默认图片生成参数、音乐生成参数、视频生成参数。
- 生成参数由用户本次生成输入传入，或使用 Tool/模型供应商默认值。
- 停用模型不能被选择。

## 异常场景

| 场景 | 触发条件 | 用户提示 | 系统行为 |
| --- | --- | --- | --- |
| 无可选模型 | 当前生成类型没有启用模型 | 当前暂无可用模型 | 阻止生成 |
| 模型已停用 | 用户选择后模型被停用 | 该模型当前不可用，请重新选择 | 要求重新选择 |
| 未选择模型 | 用户未主动选择模型 | 使用默认模型 | 使用平台默认模型进入预估 |
| 提示词安全评估不通过 | LLM 评估提示词不安全 | 提示词不符合平台规则，请修改后重试 | 不进入积分预估 |
| 确认前修改模型 | 用户进入扣费确认后尝试修改模型 | 请取消后重新选择模型 | 不允许直接修改 |
| 模型调用失败 | 供应商失败或超时 | 生成失败，可稍后重试 | 释放冻结积分由业务服务处理 |
| 参数缺失 | 缺少生成必要参数 | 请补充必要信息 | 不进入扣费确认 |

## 非目标

- 第一阶段不做长期用户模型偏好。
- 第一阶段不让用户选择文本模型。
- 第一阶段不允许进入扣费确认后直接修改模型。
- 第一阶段不向用户展示模型详细说明、供应商或内部成本。
- 第一阶段不做模型参数后台默认配置。
- 第一阶段不做语音模型选择。
- 第一阶段不把模型选择做成独立 A2UI 组件。
